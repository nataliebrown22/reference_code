# Code
## Folders
### [notebooks](notebooks)
Contains Python Jupyter notebooks (.ipynb)

Files should be prefixed in the order they should run, ie.
- 00_file_1.ipynb
- 01_file_1.ipynb
- 02_file_1.ipynb

### [scripts](scripts)
Contains Python scripts (.py)

Files should be prefixed in the order they should run, ie.
- 00_file_1.py
- 01_file_1.py
- 02_file_1.py
